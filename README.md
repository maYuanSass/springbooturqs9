# 一、系统说明

基于springboot+vue的智能停车计费系统,系统功能齐全, 代码简洁易懂，适合小白学编程,课程设计，毕业设计。

# 二、系统架构

######      前端：vue| elementui

######      后端：springboot | mybatis 

######      环境：jdk1.8+ | mysql8.0 | maven
